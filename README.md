## Sammy's Dotfiles

I run Pop!_OS for now, but I am planning to make the switch back to arch (which I currently only run VMs of), but this should work just fine as long as you have the required programs installed. I use GNU stow to manage my dotfiles, and to make it easier, I've seperated my themes into branches so that everything can be changed easily.

#### Installation

Just clone this repo, choose the branch you want, and then use GNU stow on the folders you want to use the config for.
